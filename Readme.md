# AI Warehouse Incident Reporting System

This project predicts warehouse safety incidents using AI and supports multilingual reporting.

## Features
- Incident prediction using Machine Learning
- Multi-language support
- Database storage
- User-friendly interface

## Technologies Used
- Python
- Flask
- SQLite
- Deep Translator
- Langdetect

## How to Run

1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Add your API key in the code:
   Replace YOUR_API_KEY_HERE
4. Run:
   python app.py

## Important
- Do not upload personal API keys
- Replace API key with YOUR_API_KEY_HERE
