# Warehouse AI Incident Reporter

This project detects warehouse incidents using AI and sends alerts.

## Features
- Multilingual incident reporting
- AI severity detection
- Telegram alert system
- SQLite database storage
- Flask web interface

## Technologies Used
- Python
- Flask
- SQLite
- Deep Translator
- Langdetect
- Gunicorn
- Render (deployment)

## How to Run

Install requirements:

pip install -r requirements.txt

Run the app:

python backend/app.py
